01 - INSTALAR Ambiente Virtual VENV (No Terminal VSCode PowerShell) => pip install virtualenv 02 - CRIAR e Nomear o Ambiente Virtual (Venv ou outro Nome) => python -m venv nome_do_ambiente_escolhido #Atenção - Se o Ambiente for Ativado dentro da Pasta Que foi criada anteriormente, deve entrar na pasta primeiro com comando: => cd .\cadastro_curso_womakers 03 - ATIVAR o Ambiente Virtual Venv => venv\Scripts\activate ou => .\Scripts\activate 04 - DESATIVAR o ambiente Virtual Venv => deactivate Atenção -> Problemas com PIP - Para reinstalar: => python -m ensurepip --upgrade
